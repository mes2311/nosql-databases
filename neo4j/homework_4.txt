2 EXAMPLE QUERIES:

Find all parent child relationships

MATCH (parent)-[r:FATHER_OF|:MOTHER_OF]->(child)
RETURN parent.name, type(r), child.name

Find all pilots that completed the Kessel Run and print their names,
spaceship name and the time of completing the run

MATCH (pilot)-[r:PILOTS]->(ship)-[t:COMPLETED]->(route)
WHERE route.name = 'Kessel Run'
RETURN pilot.name, type(r), ship.name, route.name, t.time
